Задача №2

Чи буде помилка при виконанні наступної програми

  • const userName = 20
  • userName = "Іштван"
Відповідь: так, буде. навідміну від змінної Let, при оголошенні conts її дані неможна змінити. В консолі з'явиться помилка.

Задача №3

Що потрапить в консоль??

  • let numOne = 20
  • let numTwo = "20"
  • let numSumm = numOne + numTwo console.log(numSumm)
Відповідь: в консоль потрапить рядок 2020. "+" це єдиний оператор який не змінує тип данних при його застосуванні (всі інші оператори змінюють тип данних на number).В результаті отримуємо додавання рідка і числа. При умові якщо хоча б одна змінна при додаванні є рядком - результат буде теж рядком.
