Задача №1

Що потрапить в консоль?

  • let someVar = 0;
  • ++someVar;
  • if (someVar) { console.log(someVar); }
Відповідь: Оскільки в цьому випадку інкримент (++) збільшує змінну на 1. Відповідно 0 збільшуємо на 1 отримуємо 1.

Задача №3

Що потрапить в консоль??

  • if (2 * 20 <= 10 || 30 / 2 < 5 && 10 <="10" || 20==="20" ) { console.log('000'); }
Відповідь: в консоль не потрапить нічого, оскільки в умові присутній логічний опертор (&& - "та") він має більший пріоретет між логічними операторами і повертає true лише тоді коли два значення будуть true. В нашому випадку 15 < 5 це брехня і дія на цьому етапі завершується.
